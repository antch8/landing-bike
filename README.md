### About:
Here is a fully responsive landing page, created for:
  - large screen `2560px`
  - wide screen `1600px`
  - laptop `1024px`
  - tablet `640px`
  - mobile (`> 320px`)

### Stack:
- HTML5
- CSS3 | Sass (SCSS)
- BEM methodology
- Flex | Grid

### Dependencies
* `node` v14.18.2
* `npm` v6.14.15

### How to start
* Fork and clone this repo
* Run `npm install`
* Run `npm start`
